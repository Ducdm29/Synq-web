Text and search field. Focus shows a mint ring.

```jsx
<Input variant="search" placeholder="Bài hát, nghệ sĩ, album…"
  leadingIcon={<i data-lucide="search" style={{width:18,height:18}}></i>} />
```

`variant="search"` rounds to a pill — the standard Nocturne search box. Pass Lucide nodes to `leadingIcon`/`trailingIcon`.
